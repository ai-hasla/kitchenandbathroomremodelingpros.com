### Hub-and-Spoke модель

**Service Hubs (Pillar Pages):** 2500-4000 слов, ссылки на все spoke pages
- Kitchen Remodeling Hub → cabinet refacing, countertops, layout design, cost guide, blog posts
- Bathroom Remodeling Hub → shower, bathtub, vanity, tile, cost guide, blog posts

**Location Hubs:** обзор всех услуг в городе, ссылки на service+location combos

**Правила внутренней перелинковки:**
- Spoke → Hub (обязательно)
- Hub → все Spokes (обязательно)
- Spoke ↔ Spoke внутри одного silo (рекомендуется)
- Hub ↔ Hub между silos (рекомендуется)
- Каждая страница достижима за ≤3 клика от главной

## 7. КОНТЕНТ НА КАЖДОЙ LOCATION PAGE (чтобы избежать дублей)

Каждая location page ОБЯЗАНА иметь **60-70% уникального контента:**

1. **Специфика жилого фонда** — типичные дома в районе, год постройки, архитектурный стиль
2. **Локальные требования к разрешениям** — permits, коды, контакты местного отдела
3. **Кейсы/проекты из этого района** — фото до/после, описание, бюджет
4. **Отзывы от клиентов из района** — с упоминанием локации
5. **Местные ориентиры** — "рядом с Green Lake", "в районе downtown Kirkland"
6. **Данные о домах** — средняя стоимость, популярные проекты ремоделинга
7. **Климат и особенности PNW** — влажность, энергоэффективность
8. **Уникальный H1, title, meta description, alt текст изображений**

## 8. ТЕХНИЧЕСКОЕ SEO

### Core Web Vitals (целевые)
- LCP < 2.5s (Astro даёт это из коробки)
- INP < 200ms (минимум JS)
- CLS < 0.1 (правильные размеры изображений)

### Schema Markup (JSON-LD на каждой странице)
```
Homepage:        HomeAndConstructionBusiness + BreadcrumbList
Service Pages:   Service + BreadcrumbList + HowTo + FAQPage
Location Pages:  LocalBusiness + BreadcrumbList + AggregateRating
Cost Guides:     Article + BreadcrumbList + FAQPage + HowTo
Blog Posts:      Article + BreadcrumbList
Gallery:         ImageGallery + BreadcrumbList
```

### Sitemaps
```
/sitemap-index.xml
  ├── /sitemap-pages.xml      (static pages)
  ├── /sitemap-services.xml   (service pages)
  ├── /sitemap-locations.xml  (location + combo pages)
  ├── /sitemap-blog.xml       (blog posts)
  └── /sitemap-gallery.xml    (gallery pages)
```

### Обязательные мета-теги на каждой странице
- Уникальный title (50-60 символов) с ключевым словом + город
- Уникальный meta description (150-160 символов) с CTA
- Open Graph теги для соцсетей
- Canonical URL
- Hreflang (не нужен, один язык)
- Robots meta (index, follow)
